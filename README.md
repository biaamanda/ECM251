# 🚀 Formação Java & Web

Este espaço é dedicado ao registro da minha jornada, cobrindo desde os fundamentos da Engenharia de Software até o desenvolvimento de aplicações web completas.

## 📌 Visão Geral do Curso
Este repositório contém exercícios, projetos e anotações baseados em uma ementa voltada para o ecossistema Java e Tecnologias Web.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem Principal:** Java (JDK)
* **IDEs:** Eclipse / NetBeans
* **Frontend:** HTML5, CSS3, JavaScript (JQuery, Bootstrap)
* **Banco de Dados:** JDBC (Java Database Connectivity)
* **Interface Gráfica:** JavaFX

## 📂 Estrutura de Aprendizado

### 1. Fundamentos e Sintaxe Java
* [ ] **Lógica de Programação:** Tipos primitivos, operadores e expressões.
* [ ] **Estruturas de Controle:** Condicionais (`if/else`) e Repetição (`for`, `while`).
* [ ] **Estruturas de Dados:** Variáveis indexadas (Arrays) e Wrappers.

### 2. Programação Orientada a Objetos (POO)
* [ ] **Conceitos Base:** Classes, atributos, métodos e construtores.
* [ ] **Pilares da POO:** Encapsulamento, Herança e Polimorfismo.
* [ ] **Abstração:** Interfaces e Classes Abstratas.
* [ ] **Tratamento de Exceções:** Gestão de erros com `try-catch`.

### 3. Java Avançado e Persistência
* [ ] **Collections:** Manipulação de listas, sets e maps.
* [ ] **Manipulação de Arquivos:** Leitura e escrita de dados (I/O).
* [ ] **Concorrência e Redes:** Threads, programação paralela e sockets.
* [ ] **Persistência de Dados:** Integração com bancos de dados via **JDBC**.

### 4. Interfaces e Web
* [ ] **Desktop GUI:** Criação de interfaces com JavaFX.
* [ ] **Frontend Moderno:** Estruturação (HTML), Estilização (CSS/Bootstrap) e Comportamento (JS).
* [ ] **Arquitetura Web:** Modelo Cliente-Servidor e introdução ao desenvolvimento Back-end.
